Full featured Odoo based Asterisk PBX solution!

Deploy and manage your phone network with joy!

Features

  • A special "package" application that contains main asterisk related modules.
  • Call tracking, SIP users management, dialplan management, call queues, and all endless Asterisk features without limits!
  • An easy to use and extend telephony platform based on Asterisk.
  • Asterisk 16 certified!
  • Typical PBX configuration templates coming soon....
